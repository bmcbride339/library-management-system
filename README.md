# Library Management System

A Java-based library management system built using object-oriented programming principles.

## Features
- User authentication and profile management
- Admin controls for managing users and inventory
- Checkout, return, renewal, and late penalty tracking
- Search functionality for library items
- Support for multiple media types (books, magazines, newspapers, audiobooks)
- Persistent file-based storage

## Technologies Used
- Java
- Object-Oriented Programming (Inheritance, Polymorphism)
- File I/O

## Purpose
Developed as an academic project to demonstrate system design, OOP concepts, and real-world problem solving.
